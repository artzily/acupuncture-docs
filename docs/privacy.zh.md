# Face Meridian AI 隐私政策
在 Face Meridian AI，我们最重视的事项之一就是访客的隐私。本隐私政策文件说明了 Face Meridian AI 收集和记录的信息类型，以及我们如何使用这些信息。

如果您有其他问题，或需要了解有关我们隐私政策的更多信息，请随时与我们联系。

本隐私政策仅适用于我们的线上活动，并适用于访问我们系统的访客，涉及他们在 Face Meridian AI 中共享和/或被收集的信息。本政策不适用于线下收集的信息，或通过我们系统以外渠道收集的任何信息。

## 同意
使用我们的系统即表示您同意我们的隐私政策并接受其条款。

## 我们收集的信息
当我们要求您提供个人信息时，我们会在要求提供信息的时点清楚说明需要您提供哪些个人信息，以及要求提供这些信息的原因。

如果您直接联系我们，我们可能会收到关于您的其他信息，例如您的姓名、电子邮箱地址、电话号码、您发送给我们的消息内容和/或附件，以及您选择提供的任何其他信息。

当您注册账户时，我们可能会要求您提供联系信息，包括您的姓名、头像以及部分个人信息等内容。

## 面部数据收集
Face Meridian AI 不会将用户的面部数据采集、存储或传输到我们的服务器。所有面部分析均在用户设备本地完成，分析结果会直接展示给用户。

## AI 聊天功能
如果您使用 AI 聊天功能，我们会处理您提交的文本，以便为您生成回复。

为提供此功能，您的聊天输入会被传输至 OpenAI API（ChatGPT API），生成的回复将通过我们的应用直接返回给您。除为处理请求并交付回复所必需的范围外，我们不会在自己的后端或应用数据库中存储您的聊天提示词或回复内容。

OpenAI 可能会根据其自身政策处理和保留 API 数据。OpenAI 已发布的 API 文档说明，默认情况下，API 输入和输出不会用于训练 OpenAI 模型，但某些 API 数据可能会为滥用监测而保留最多 30 天，除非适用不同的数据控制措施。

## 我们如何使用您的信息
我们以多种方式使用所收集的信息，包括：

提供、运营和维护我们的系统
改进、个性化和扩展我们的系统
了解并分析您如何使用我们的系统
开发新的产品、服务、功能和特性
根据您提交的文本生成 AI 聊天回复
与您沟通，无论是直接沟通还是通过我们的合作伙伴沟通，包括用于客户服务、向您提供与系统相关的更新和其他信息，以及用于营销和推广目的
向您发送电子邮件
发现并防止欺诈

# 日志文件
Face Meridian AI 遵循使用日志文件的标准流程。这些文件会在访客使用我们的系统时记录相关信息。所有托管公司都会这样做，这也是托管服务分析的一部分。日志文件收集的信息包括互联网协议（IP）地址、浏览器类型、互联网服务提供商（ISP）、日期和时间戳、引用/退出页面，以及可能的点击次数。这些信息不会与任何可识别个人身份的信息相关联。收集这些信息的目的是分析趋势、管理网站、追踪用户在系统中的活动情况，并收集人口统计信息。

## 广告合作伙伴隐私政策
您可以查阅此列表，以找到 Face Meridian AI 各广告合作伙伴的隐私政策。

第三方广告服务器或广告网络会使用 Cookie、JavaScript 或网络信标等技术，这些技术会用于其各自出现在 Face Meridian AI 上的广告和链接，并直接发送到用户的浏览器。当这种情况发生时，它们会自动接收您的 IP 地址。这些技术用于衡量其广告活动的效果和/或个性化您在所访问系统中看到的广告内容。

请注意，Face Meridian AI 无法访问或控制这些由第三方广告商使用的 Cookie。

## 第三方隐私政策
Face Meridian AI 的隐私政策不适用于其他广告商或网站。因此，我们建议您查阅这些第三方广告服务器各自的隐私政策，以获取更详细的信息。其中可能包括其做法，以及关于如何选择退出某些选项的说明。

您可以通过各自浏览器的设置选择禁用 Cookie。有关特定浏览器 Cookie 管理的更多详细信息，可在相应浏览器的网站上找到。

## CCPA 隐私权（请勿出售我的个人信息）
根据 CCPA，除其他权利外，加利福尼亚州消费者享有以下权利：

要求收集消费者个人数据的企业披露其所收集的个人数据类别以及具体的个人数据信息。

要求企业删除其所收集的有关该消费者的任何个人数据。

要求出售消费者个人数据的企业不要出售该消费者的个人数据。

如果您提出请求，我们将在一个月内回复您。如果您想行使这些权利，请与我们联系。

## GDPR 数据保护权利
我们希望确保您充分了解自己拥有的所有数据保护权利。每位用户均享有以下权利：

访问权：您有权请求获取您的个人数据副本。我们可能会为此服务收取少量费用。

更正权：您有权要求我们更正您认为不准确的任何信息。您也有权要求我们补充您认为不完整的信息。

删除权：在某些条件下，您有权要求我们删除您的个人数据。

限制处理权：在某些条件下，您有权要求我们限制对您个人数据的处理。

反对处理权：在某些条件下，您有权反对我们处理您的个人数据。

数据可携带权：在某些条件下，您有权要求我们将所收集的数据传输给其他组织，或直接传输给您本人。

如果您提出请求，我们将在一个月内回复您。如果您想行使这些权利，请与我们联系。

## 儿童信息
我们工作的另一项重点是在儿童使用互联网时加强对他们的保护。我们鼓励家长和监护人观察、参与、监督并指导儿童的线上活动。

Face Meridian AI 不会故意收集 13 岁以下儿童的任何个人身份信息。如果您认为您的孩子在我们的系统中提供了此类信息，我们强烈建议您立即与我们联系，我们将尽最大努力尽快从记录中删除这些信息。
